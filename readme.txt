=== Zocdoc Booking Widget Plugin ===
Contributors: alexdoynezocdoc

This plugin allows practices who are customer\'s of Zocdoc to easily integrate Zocdoc\'s online booking into their website. Please reach out to your Zocdoc account manager or service@zocdoc.com to be set up!

== Installation ==
After enabling the plugin simply add your API Key, and the names and locations of the providers in the order that you\'d like them to show up on your website.
